# _Sharly Chess_ release notes

## Documents

- Added a new document to get a summary of norms achieved, or a forecast of norms that can be achieved for the final round (4.2.0)
- Add options to the norm document for the number of rounds and the event type (4.2.0)
- Added detailed calculations to the norm report to allow for easy validation (4.2.0)
- Added automatic removal of specific played games in order to try to fulfil the norm requirements (4.2.0)
- Fix a norm calculation bug that incorrectly considered CM and WCM as title-holders (4.2.0)
- Added support for norm exemption 1.4.2c (last round forfeit as loss) (4.2.0)

## Network

- Improved detection of internet connectivity (4.1.5)
