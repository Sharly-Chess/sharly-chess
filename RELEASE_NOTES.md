# _Sharly Chess_ release notes

## General

- Improved detection of internet connectivity (4.2.0)
- Prevent reloading templates (4.2.1)

## Tournaments

- TRF26 import / export (4.2.0)
- Fixed player distribution by criteria (4.2.3)

## Players

- Clarify actions on absent players if over half absent players (4.2.1)
- Fixed `Withdraw from tournament` button absent from the record modal (4.2.4)

## Documents

- Added a new document to get a summary of norms achieved, or a forecast of norms that can be achieved for the final round (4.2.0)
- Added options to the norm document for the number of rounds and the event type (4.2.0)
- Added detailed calculations to the norm report to allow for easy validation (4.2.0)
- Added automatic removal of specific played games in order to try to fulfil the norm requirements (4.2.0)
- Fixed a norm calculation bug that incorrectly considered CM and WCM as title-holders (4.2.0)
- Added support for norm exemption 1.4.2c (last round forfeit as loss) (4.2.0)
- Added custom team ranking documents (4.2.0)
- For norm calculations, 1.4.3 exemption now also waive 1.4.4 as confirmed to us by the FIDE (4.2.1)
- For norm calculations, players under the FID federation are now disregarded when considering foreign federation counts (4.2.1)

## Screens

- Restored check-in on input screens (4.2.0)
- Fixed default global background color (4.2.1)
- Fixed display controller redirection on results or check-in input (4.2.1)
- Escape special characters in timer texts (4.2.3)
- Fixed players' display on players screens before pairing (4.2.3)

## _Sharly-Chess.com_

- Added missing timeouts (4.2.0)
- Prevent moving players from a synced tournament to a not-synced tournament (4.2.0)
- Fixed tournament import of schools tournaments (4.2.1)

## _FFE_

- Clarified the rules upload (4.2.0)
