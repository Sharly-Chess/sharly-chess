# Français
Ce dossier est le dossier par défaut des fichiers Papi utilisés par les tournois dans Sharly-Chess.
Cela peut être configuré à la fois au niveau de l'évènement et du tournoi.

# English
This directory is the default location for the Papi files used by the tournaments is Sharly-Chess.
It can be overridden at event and tournament level.
