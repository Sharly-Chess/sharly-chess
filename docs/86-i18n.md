# Papi-web - Internationalization

## References

- [Babel documentation](https://babel.pocoo.org/en/stable/)

- [Jinja2 / Babel integration](https://github.com/macagua/python_i18n_babel_jinja2)

## Files

| File                                     | Format           | Usage                                                                   |
|------------------------------------------|------------------|-------------------------------------------------------------------------|
| `/utils/i18n_update.py`                  | text / Python    | The main script to manage translations.                                 |
| `/utils/babel.cfg`                       | text / INI       | The configuration used by Babel to extraction the strings to translate. |
| `/locale/messages.pot`                   | text / gettext   | The Portable Object Template, common to all the translations.           |
| `/locale/<lang>/LC_MESSAGES/messages.po` | text / gettext   | The Portable Object of translation `<lang>`.                            |
| `/locale/<lang>/LC_MESSAGES/messages.mo` | binary / gettext | The Machine Object of translation `<lang>`.                             |

## Management

The script `utils/i18n_update.py` does all the tasks needed to manage the translations.

### Extracting strings from the sources 

Thanks to the configuration defined in [`utils/babel.cfg`](../utils/babel.cfg), strings to translate are extracted from the sources to the PO template:

- `src/**.py`, `src/web/templates/*.html` > `locale/messages.pot`

### Creating PO files for new languages

Needed folders and files are created for new languages:

- `utils/i18n_update.LOCALES` > `locale/<locale>/LC_MESSAGES/messages.po`

To add a new language, update variable `LOCALES` in `utils/i18n_update.py` (and later add the translations to the PO file!).

### Update PO files

Source modifications are reported to the PO files:

- `locale/messages.pot` > `locale/<locale>/LC_MESSAGES/messages.po`

### Compiling PO files to MO files

The MO binary files (used at run-time) are compiled from the PO text files:

- `locale/<locale>/LC_MESSAGES/messages.po` > `locale/<locale>/LC_MESSAGES/messages.mo`
