# Sharly Chess - Raccourcis clavier

Cette page est un document de travail sur les raccourcis clavier utilisables dans l'application, à destination des développeur·euses.

## Onglet Joueur·euses

### Raccourcis globaux

| Raccourci Papi | Action                   | Sharly Chess                                 |
|----------------|--------------------------|----------------------------------------------|
| ``F3``         | ~~Ajouter un·e joueur·euse~~ | Non implémentée                              |
| ``F5``         | ~~Créer un·e joueur·euse~~   | Non implémenté (actualisation du navigateur) |

## Onglet Appariements

### Partie gauche (joueur·euses non apparié·es)

Toutes les actions ne sont pas possibles en fonction de l'état du·de la joueur·euse.

| Raccourci Papi | Action                                   | Sharly Chess   |
|----------------|------------------------------------------|----------------|
| ``C``          | Copier                                   | Non implémenté |
| ``V``          | Coller                                   | Non implémenté |
| ``M``          | Modifier                                 | Non implémenté |
| ``Suppr``      | Suppression                              | Non implémenté |
|                | Retour ronde en cours                    | Non implémenté |
| ``R``          | Retour définitif                         | Non implémenté |
|                | Forfait ronde en cours                   | Non implémenté |
| ``F``          | Forfait définitif                        | Non implémenté |
| ``4``          | Point joker                              | Non implémenté |
| ``6``          | Demi point joker                         | Non implémenté |
| ``0``          | Annuler joker                            | Non implémenté |
| ``=``          | Bye                                      | Non implémenté |
| ``B``          | Apparier Blancs                          | Non implémenté |
| ``N``          | Apparier Noirs                           | Non implémenté |
| ``X``          | Apparier Exempt                          | Non implémenté |
| ``Down``       | Joueur·euse suivant·e                    | Non implémenté |
| ``Up``         | Joueur·euse précédent·e                  | Non implémenté |

### Partie droite (échiquiers)

|    Papi    | Action                                          | Sharly Chess   |
|:----------:|-------------------------------------------------|----------------|
|   ``0``    | **Pas de résultat**                             | ``0``          |
|   ``1``    | **Gain Blancs**                                 | ``1``          |
|   ``2``    | **Gain Noirs**                                  | ``2``          |
|   ``3``    | **Nulle**                                       | ``3``          |
|   ``P``    | Permuter                                        | Non implémenté |
|   ``D``    | Désapparier                                     | Non implémenté |
|   ``4``    | Gain Blancs forfait                             | Non implémenté |
|   ``5``    | Gain Noirs forfait                              | Non implémenté |
|   ``7``    | Double forfait                                  | Non implémenté |
|  ``Down``  | **Échiquier suivant**                           | ``Down``       |
|   ``Up``   | **Échiquier précédent**                         | ``Up``         |
